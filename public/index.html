MANGAP API

Documentation

BASE URL = http://localhost:5000

Hosting atau lebih bagus jalankan di localhost

Recommended Comic

/recommended

Example: http://localhost:5000/recommended

Popular Comic

/popular

Example: http://localhost:5000/popular

List Comic

/daftar-komik/:page

Example: http://localhost:5000/daftar-komik/1

Detail Comic

/detail/[endpoint]

Example: http://localhost:5000/detail/solo-leveling

Search Comic

/search?keyword=[keyword]

Example: http://localhost:5000/search?keyword=solo

Read Comic

/read/[endpoint]

Example: http://localhost:5000/read/solo-leveling-chapter-33-bahasa-indonesia/

Genre

/genre

Example: http://localhost:5000/genre

Genre Comic

/genre/[endpoint]

Example: http://localhost:5000/genre/action?page=1
